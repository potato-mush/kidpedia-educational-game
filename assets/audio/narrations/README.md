## How to Create Narration Audio

### Free Text-to-Speech Services:
- **TTSMaker** (https://ttsmaker.com/) - Free, no login required
- **Natural Readers** (https://naturalreaders.com/)
- **Google Cloud TTS** (https://cloud.google.com/text-to-speech)

### Audio Specifications:
- **Format**: MP3
- **Bitrate**: 128 kbps or higher
- **Sample Rate**: 44.1 kHz
- **Duration**: 30-60 seconds per file
- **Voice**: Child-friendly, clear pronunciation
