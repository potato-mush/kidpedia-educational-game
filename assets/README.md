# Asset Placeholders

This directory contains placeholder files to maintain the folder structure.

## Required Asset Files

### Images
Add educational images to:
- `animals/` - Animal photos
- `space/` - Space and astronomy images
- `science/` - Science illustrations
- `history/` - Historical images
- `geography/` - Geographic images
- `badges/` - Badge icons

### Videos
Add educational videos to:
- `videos/animals/` - Animal videos
- `videos/space/` - Space videos
- `videos/science/` - Science videos

### Audio
Add audio files to:
- `audio/narrations/` - Topic narrations
- `audio/sounds/` - Sound effects and animal sounds

### Animations
Add Lottie JSON files to:
- `lottie/` - Animation files

### Fonts
Add Poppins font family to:
- `fonts/` - TrueType font files

## Notes
- Assets are referenced in the app but need to be added manually
- Ensure file paths match those defined in `seed_data_service.dart`
- Optimize images and videos for mobile before adding
- Keep file sizes reasonable for offline use
