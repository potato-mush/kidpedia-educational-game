## Video Specifications:
- **Format**: MP4 (H.264 codec)
- **Resolution**: 720p (1280x720) or 1080p (1920x1080)
- **Duration**: Full video plays to completion (30-60 seconds recommended)
- **Size**: Under 10MB per file for web compatibility
- **Frame Rate**: 30 fps
- **Aspect Ratio**: 16:9
